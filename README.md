# FLY - P2P Multiplayer Flight Game

fly.pieter.com 스타일의 3D 멀티플레이어 비행 게임입니다.
PeerJS를 사용한 P2P 연결로 서버 없이 멀티플레이어를 지원합니다.

## Features

- 3D 비행 시뮬레이터 (Three.js)
- P2P 멀티플레이어 (PeerJS)
- 방 생성/참가 시스템
- 3가지 비행체 선택 (Cessna, Fighter Jet, Cyberpink)
- 실시간 채팅
- 미니맵 & 리더보드
- 모바일 터치 컨트롤 지원
- 광고 배너 공간 포함

## Controls

### Desktop
- **Arrow Keys** - 비행 조종 (좌/우 회전)
- **W/S** - 스로틀 (가속/감속)
- **Q/E** - 피치 (상승/하강)
- **Space** - 미사일 발사
- **V** - 카메라 전환
- **Enter** - 채팅

### Mobile
- **Left Joystick** - 비행 조종
- **Right Joystick** - 스로틀
- **FIRE** - 발사
- **BOOST** - 부스트
